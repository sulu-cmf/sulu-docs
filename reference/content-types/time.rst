Time
====

Description
-----------

Shows a text line, the inserted content will be validated against a localized
time string and saved as a simple string.

Parameters
----------

No parameters available

Example
-------

.. code-block:: xml

    <property name="time" type="time">
        <meta>
            <title lang="en">Time</title>
        </meta>
    </property>

Twig
----

.. code-block:: twig

    {{ content.time }}
