Phone
=====

Description
-----------

Shows a text line, the inserted content will be saved as a simple string.

Parameters
----------

No parameters available

Example
-------

.. code-block:: xml

    <property name="phone" type="phone">
        <meta>
            <title lang="en">Phone number</title>
        </meta>
    </property>

Twig
----

.. code-block:: twig

    {{ content.phone }}
