Text area
=========

Description
-----------

Shows a simple text area, the inserted content will be saved as simple string.

Parameters
----------

No parameters available

Example
-------

.. code-block:: xml

    <property name="description" type="text_area">
        <meta>
            <title lang="en">Description</title>
        </meta>
    </property>

Twig
----

.. code-block:: twig

    {{ content.description }}
